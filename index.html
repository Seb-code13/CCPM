GanttProject Tutorial

Why using CCPM ?

It's a project management method that basically focuses on managing deadlines and activities, considering the allocation of resources, based on the Theory of Constraints (TOC).
CCPM avoids multi-tasking employees. After study, we noticed that an employee dedicated to a single task is more efficient. (We can see this from the SMED method, which is used in all areas to reduce the time taken to complete an activity).
In addition, the CCPM optimises project time and employee efficiency.
In this article, we will look at the steps to follow.

Critical Current Chain Method

Once you have planned your various projects on Ganttproject, make sure that the critical path has been completed for each project.
Find the critical resource (it's a person) and put it forward.

For exemple this is 2 project of Creation of Website, planificated on GanttProject, every color means a different ressource.
We can see both project have the same aspect and resources are multitasking.
First as you can see the red color is the critical resources because it often appears.

Now, you need to adapt the different projects according to the Critical Chain Method.
Take half the time allocated for each task and accumulate it at the end as 'buffer time' (pink color).

Now you need to adapt the different projects according to the critical resource.
You need to shift the project start date to take account of the critical resource. You have to avoid multi-tasking on the critical resource.
